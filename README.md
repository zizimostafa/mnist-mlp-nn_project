MNIST Handwritten Digit Recognition using MLP

Project Description:
This project implements a Multilayer Perceptron (MLP) using PyTorch to classify handwritten digits from the MNIST dataset (0–9).

Model:
- Input Layer: 784 neurons
- Hidden Layers: 128, 64 neurons
- Output Layer: 10 classes

Experiments:
- ReLU activation
- Tanh activation

Results:
- ReLU Accuracy: 97.85%

- Tanh Accuracy:97.50%

Tools:
- PyTorch
- torchvision
- matplotlib
